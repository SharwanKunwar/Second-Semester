## Wap to calculate sum of the natural number up to N number using recursive function.

```c
#include<stdio.h>

int fun(int size){
    if(size == 0){
        return 0;
    }
    return size+fun(size-1);
}

int main(){
    int size; 

    printf("Enter the size : ");
    scanf("%d",&size);

    int result = fun(size);
    printf("%d",result);

    return 0;

}
```
